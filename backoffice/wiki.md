## 3. Principes techniques (conseil client)

### Fonctionnement

- Produit configurable via l’application ou le portail client.

### Alimentation

- Alimentation selon le modèle (batterie rechargeable ou secteur).

- Usage en intérieur recommandé ; éviter humidité et chocs.
- Température de fonctionnement : plage indiquée sur la notice du produit.

### Connectivité

- Configuration initiale guidée dans l’application (assistant de première utilisation).

## 8. Dépannage rapide

| Symptôme | Cause fréquente | Action |
|----------|-----------------|--------|
| Produit ne répond plus | Mise en veille ou batterie faible | Redémarrer ; recharger si applicable |
| Ne se connecte plus au réseau | Changement box / mot de passe Wi‑Fi | Reconfigurer via l’application |
| Application ne trouve pas le produit | Produit hors ligne ou mauvais réseau | Vérifier Wi‑Fi ; rapprocher du routeur |
| Contenu non à jour | Synchronisation en attente | Forcer la synchronisation dans l’app ; redémarrer le produit |
| Problème persistant | Cause variable | Contacter le support avec numéro de série et description du symptôme |

**Réinitialisation usine** : procédure décrite dans la notice ou l’application — efface la configuration réseau et le lien compte, sans supprimer les données côté cloud (à resynchroniser).
