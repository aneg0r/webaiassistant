Comment ça marche

📎

Intégrez le widget

Une ligne CSS, un script : le bouton flottant s’affiche sur votre site sans refonte.

⚙️

Configurez le périmètre

FAQ, dépannage, SAV, rappel, newsletter : activez uniquement les scénarios utiles.

🤝

Reprenez la main

L’admin permet de répondre en direct et de bloquer l’IA le temps du traitement.

✨

Pensé pour le commerce

  • Réponses fondées sur votre FAQ — pas d’invention hors sujet
  • Scénarios multi-tours : dépannage, commande, retours SAV
  • Demande de rappel et inscription newsletter enregistrées côté serveur
  • Transcriptions JSON pour le support et l’amélioration continue
  • Interface admin : sessions, archivage, indicateur « en ligne »

Intégration en quelques minutes

Hébergez l’API derrière nginx, copiez backoffice/perimeter.json selon votre offre (produit physique ou service), puis ajoutez le widget sur vos pages. 

<link rel="stylesheet" href="/static/widget/chat-widget.css" />
<div id="ixfeedback-widget-root"></div>
<!-- chat HTML: #ixchat-widget-root — voir index.html -->
<script>
  window.CHAT_API_URL = "/agent/prompt";
  window.FEEDBACK_PAGE_API_URL = "/agent/feedback-page";
</script>
<script src="/static/agent.js"></script>
<script src="/static/widget/chat-widget.js"></script>
